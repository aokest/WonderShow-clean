# WonderShow 靈演社群版

靈演社群版是面向創作者、講師和開發者的免費可用版本。2.0 社群版提供穩定、離線、可重現的簡報錄製與輕量後期工作流，讓使用者可以在 macOS 上錄製簡報視窗、講者攝影機和麥克風聲音，保存 `.wondershow` 專案，並在錄製後完成基礎鏡頭編輯、片段選取和匯出。

## 功能範圍

社群版包含：

- 攝影機接入：支援 Mac 內建攝影機、常見 UVC 攝影機和外接擷取設備。
- 螢幕和視窗錄製：可錄製簡報視窗、整個螢幕或手動選擇的視窗。
- 錄製中切源：支援 `⌘1` 到 `⌘6` 切換源位。
- 講者子母畫面和基礎版面：螢幕主畫面、講者子母畫面、講者主畫面、左右分屏、上下分屏、純螢幕、純講者。
- 多畫布比例：橫屏、直屏、方屏，並能在預覽/匯出中重現錄製時的版面和畫布變化。
- 音訊錄製：跟隨 macOS 目前麥克風輸入，生成獨立原始音訊軌。
- 專案化保存：生成 `.wondershow` 專案結構，保留原始軌、合成軌、時間軸、版面和匯出資訊。
- 合成預覽與匯出：錄製後預覽合成影片，匯出 MP4/MOV/GIF，手動匯出最高支援 4K。
- 基礎鏡頭編輯：矩形/圓形螢幕鏡頭選區、自動關鍵幀、標準遮罩、基礎邊框、復原/重做、單片段和少量片段合併匯出。
- 基礎講者畫面調整：鏡像、亮度、對比度和輕量柔化。
- 多語言介面：簡體中文、繁體中文和英文。

## 快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `⌥⌘R` | 開始 / 停止錄製 |
| `⌘1` - `⌘6` | 錄製中切換源位 |
| `⌘Z` / `⇧⌘Z` | 復原 / 重做鏡頭或片段操作 |
| `⌘←` / `⌘→` | 上一幀 / 下一幀 |
| `⌘N` | 新增鏡頭關鍵幀 |
| `Delete` | 刪除選中的關鍵幀或片段 |
| `Return` | 播放 / 暫停鏡頭預覽 |
| `ESC` | 取消目前鏡頭選區，並回復剛生成的自動關鍵幀 |

## 使用說明

1. 開啟 `靈演社群版.app`。
2. 在 macOS 系統設定中允許攝影機、麥克風和螢幕錄製權限。攝影機權限需要由 App 發起請求後才會出現在系統列表裡。
3. 在右側選擇輸入設備和音訊輸入。
4. 選擇錄製源：簡報視窗、整個螢幕或手動選擇視窗。
5. 選擇錄製模式和版面，例如「攝影機 + 螢幕」和「螢幕主畫面 + 講者子母畫面」。
6. 點擊「開始錄製」，結束後等待合成輸出。
7. 使用「預覽合成」檢查結果；需要精修時進入鏡頭編輯，完成關鍵幀或片段選取後再匯出影片。

## 開源文檔和程式碼

開源倉庫包含：

- WonderShow Core Swift Package。
- `.wondershow` 專案 manifest、時間軸、版面、匯出配置等公開 schema。
- MediaPipe sidecar 本地協議和安全約束。
- 插件 API、示例、測試。
- `README`、`LICENSE`、`NOTICE`、`CONTRIBUTING`、`SECURITY`、`ROADMAP`、`PACKAGE_BOUNDARY`。
- 社群版三語說明：簡體中文、繁體中文、英文。
