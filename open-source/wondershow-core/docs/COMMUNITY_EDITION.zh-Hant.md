# WonderShow 靈演社群版

靈演社群版是面向創作者、講師和開發者的免費可用版本。它聚焦一條清晰的錄製工作流：在 macOS 上錄製簡報視窗、講者攝影機畫面和麥克風聲音，並生成可預覽、可匯出的專案。

## 功能介紹

- 攝影機接入：支援 Mac 內建攝影機、常見 UVC 攝影機和外接擷取設備。
- 螢幕和視窗錄製：可錄製簡報視窗、整個螢幕或手動選擇的視窗。
- 講者子母畫面：支援螢幕主畫面加講者子母畫面，也支援講者主畫面、左右分屏、只錄屏、只錄講者等版面。
- 音訊錄製：跟隨 macOS 目前麥克風輸入，生成獨立原始音訊軌。
- 專案化保存：生成 `.wondershow` 專案結構，保留原始軌、合成軌、時間軸和匯出資訊。
- 合成預覽與匯出：錄製後可以預覽合成影片，並匯出最終 MP4。
- 多語言介面：內建簡體中文、繁體中文和英文介面。

## 應用場景

- 錄製課程、微課、知識分享、產品演示和線上講座。
- 給 Keynote、PowerPoint、WPS、PDF 或網頁簡報錄製講解影片。
- 為影片平台、YouTube、內部學習平台準備帶講者子母畫面的素材。
- 在正式錄課前，驗證攝影機、麥克風、螢幕錄製權限和畫面版面。
- 開發者讀取 `.wondershow` 專案格式，編寫專案檢查、轉換、歸檔或素材管理工具。

## 使用說明

1. 開啟 `靈演社群版.app`。
2. 在 macOS 系統設定中允許攝影機、麥克風和螢幕錄製權限。攝影機權限需要由 App 發起請求後才會出現在系統列表裡。
3. 在右側選擇輸入設備和音訊輸入。
4. 選擇錄製源：簡報視窗、整個螢幕或手動選擇視窗。
5. 選擇錄製模式和版面，例如「攝影機 + 螢幕」和「螢幕主畫面 + 講者子母畫面」。
6. 點擊「開始錄製」，結束後等待合成輸出。
7. 使用「預覽合成」檢查結果，再匯出影片或保留專案檔繼續處理。

## 特點說明

- 社群版強調穩定、可用和透明，介面只保留核心錄製所需的能力。
- 社群版使用獨立 bundle id `com.wondershow.community`，便於 macOS 單獨管理權限。
- 打包產物包含簽名權限聲明，便於 macOS 正確請求攝影機、麥克風和螢幕錄製權限。
- 錄製主鏈路繼承 v1.0.0 受保護基線：活動視窗直接採集、切換錄製源、完整視窗顯示、預覽/匯出一致和高清匯出。

## 開源專案適合誰

- 想了解 WonderShow 專案格式、錄製 manifest、時間軸和匯出配置的開發者。
- 想寫 `.wondershow` 專案檢查器、轉換器、批次處理工具或素材管理工具的自動化開發者。
- 想基於公開專案格式做內容歸檔、課程素材整理或內部工作流整合的團隊。
- 想學習 macOS 錄製產品如何拆分應用體驗與開放資料格式的獨立開發者。

開源包不包含完整桌面 App 源碼、ScreenCaptureKit 採集實現、即時監視器、影片合成器、商業 UI、授權系統或更新系統。它的目標是讓社群能基於公開專案格式做工具和整合，同時保持桌面應用本身的清晰邊界。
