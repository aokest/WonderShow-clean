# WonderShow 灵演社区版

灵演社区版是面向创作者、讲师和开发者的免费可用版本。2.0 社区版提供稳定、离线、可复现的演示录制与轻量后期工作流，让用户可以在 macOS 上录制演示窗口、讲者摄像头和麦克风声音，保存 `.wondershow` 项目，并在录制后完成基础镜头编辑、片段选取和导出。

## 功能范围

社区版包含：

- 摄像头接入：支持 Mac 内置摄像头、常见 UVC 摄像头和外接采集设备。
- 屏幕和窗口录制：可录制演示窗口、整个屏幕或手动选择的窗口。
- 录制中切源：支持 `⌘1` 到 `⌘6` 切换源位。
- 讲者画中画和基础布局：屏幕主画面、讲者画中画、讲者主画面、左右分屏、上下分屏、纯屏幕、纯讲者。
- 多画布比例：横屏、竖屏、方屏，并能在预览/导出中复现录制时的布局和画布变化。
- 音频录制：跟随 macOS 当前麦克风输入，生成独立原始音频轨。
- 项目化保存：生成 `.wondershow` 项目结构，保留原始轨、合成轨、时间轴、布局和导出信息。
- 合成预览与导出：录制后预览合成视频，导出 MP4/MOV/GIF，手动导出最高支持 4K。
- 基础镜头编辑：矩形/圆形屏幕镜头选区、自动关键帧、标准遮罩、基础边框、撤销/重做、单片段和少量片段合并导出。
- 基础讲者画面调整：镜像、亮度、对比度和轻量柔化。
- 多语言界面：简体中文、繁体中文和英文。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌥⌘R` | 开始 / 停止录制 |
| `⌘1` - `⌘6` | 录制中切换源位 |
| `⌘Z` / `⇧⌘Z` | 撤销 / 重做镜头或片段操作 |
| `⌘←` / `⌘→` | 上一帧 / 下一帧 |
| `⌘N` | 新建镜头关键帧 |
| `Delete` | 删除选中的关键帧或片段 |
| `Return` | 播放 / 暂停镜头预览 |
| `ESC` | 取消当前镜头选区，并回滚刚生成的自动关键帧 |

## 使用说明

1. 打开 `灵演社区版.app`。
2. 在 macOS 系统设置中允许摄像头、麦克风和屏幕录制权限。摄像头权限需要由 App 发起请求后才会出现在系统列表里。
3. 在右侧选择输入设备和音频输入。
4. 选择录制源：演示窗口、整个屏幕或手动选择窗口。
5. 选择录制模式和布局，例如“摄像头 + 屏幕”和“屏幕主画面 + 讲者画中画”。
6. 点击“开始录制”，结束后等待合成输出。
7. 使用“预览合成”检查结果；需要精修时进入镜头编辑，完成关键帧或片段选取后再导出视频。
