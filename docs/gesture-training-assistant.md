# WonderShow Gesture Training Assistant Docs

This is the language index for the standalone gesture training assistant.

- 简体中文: [gesture-training-assistant.zh-Hans.md](gesture-training-assistant.zh-Hans.md)
- 繁體中文: [gesture-training-assistant.zh-Hant.md](gesture-training-assistant.zh-Hant.md)
- English: [gesture-training-assistant.en.md](gesture-training-assistant.en.md)

The training assistant is separate from the main WonderShow app. It can collect static hand-shape samples, train a local `landmark_v2` model, and export a `.wsgesture` package. The main app import and management interface is documented in each localized version, but no main-app UI or recording code is changed here.
