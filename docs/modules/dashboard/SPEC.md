# Dashboard 规格

## 页面分区

- 顶部：产品标题、关键状态、主操作按钮
- 左侧：导播预览与叠加状态
- 右侧：
  - 快速开始
  - 演示控制
  - 手势工作区
  - 设备与录制摘要
- 底部或折叠区：高级诊断

## 必须展示的信息

- 摄像头状态
- 手势状态
- 当前演示目标
- 录制状态
- 最近一次动作
- 当前设备名
- 当前录制源、录制模式、布局、画中画参数
- 当前项目保存位置、raw 轨数量、program 输出状态
- 当前音频输入设备
- 导出进度、导出尺寸、文件大小和成功/失败反馈

## 必须保留的操作

- 刷新设备
- 打开测试演示页
- 开始/结束彩排
- 开始/暂停/继续/终止录制
- 进入个人手势校准
- 打开系统权限设置
- 选择屏幕/窗口录制源，支持列表和缩略图
- 选择麦克风输入
- 拖拽/缩放/改变画中画形状
- 录制中切换布局，并将变化写入 program timeline
- 导入项目、打开项目、预览合成、导出项目、导出视频、在 Finder 中显示

## 未来交互

- 录制源选择器支持用户为活跃窗格分配 `1` 到 `6` 的源位。
- 录制时按 `⌘1` 到 `⌘6` 可快速把对应源位切到监控台，并同步进入屏幕 raw 轨。
- mini toolbar 和菜单栏常驻模式也应暴露当前源位和快速切源入口。
