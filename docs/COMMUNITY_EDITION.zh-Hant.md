# WonderShow 靈演社群版

靈演社群版是面向創作者、講師和開發者的免費可用版本。它保留核心錄製鏈路，讓你可以在 macOS 上錄製簡報視窗、攝影機講者畫面和麥克風聲音，並生成可預覽、可匯出的影片專案。

專業版仍在開發測試中。希望在不久的未來，專業版以及其他配套工具可以和大家見面。

## 功能介紹

- 攝影機接入：支援 Mac 內建攝影機、常見 UVC 攝影機和外接擷取設備。
- 螢幕和視窗錄製：可選擇簡報視窗、整個螢幕或可錄製視窗源。
- 講者子母畫面：支援 PPT/螢幕主畫面加講者子母畫面，也支援講者主畫面、左右分屏、只錄屏、只錄講者等版面。
- 音訊錄製：跟隨 macOS 目前麥克風輸入，生成獨立原始音訊軌。
- 專案化保存：生成 `.wondershow` 專案結構，保留原始軌、合成軌、時間軸和匯出資訊。
- 合成預覽與匯出：錄製後可以預覽合成影片，並匯出最終 MP4。
- 基礎講者畫面調節：社群版保留鏡像、亮度、對比、柔化等基礎效果。
- 多語言介面：內建簡體中文、繁體中文和英文介面。

## 應用場景

- 錄製課程、微課、知識分享、產品演示和線上講座。
- 給 Keynote、PowerPoint、WPS、PDF 或網頁簡報錄製講解影片。
- 為影片平台、YouTube、內部培訓平台準備帶講者子母畫面的素材。
- 在正式直播或錄課前，驗證攝影機、麥克風、螢幕錄製權限和畫面版面。
- 開發者驗證 `.wondershow` 專案格式，編寫專案讀取、校驗、轉換或插件原型。

## 使用說明

1. 開啟 `靈演社群版.app`。
2. 在 macOS 系統設定中允許攝影機、麥克風和螢幕錄製權限。攝影機權限需要由 App 發起請求後才會出現在系統列表裡。
3. 在右側選擇輸入設備和音訊輸入。
4. 選擇錄製源：簡報視窗、整個螢幕或手動選擇視窗。
5. 選擇錄製模式和版面，例如「攝影機 + 螢幕」和「PPT主畫面 + 講者子母畫面」。
6. 點擊「開始錄製」，結束後等待合成輸出。
7. 使用「預覽合成」檢查結果，再匯出影片或保留專案檔繼續處理。

## 特點說明

- 社群版強調穩定、可用和透明，不展示 VIP、SVIP 或手勢控制入口。
- 社群版不包含正在實驗中的高級美顏、背景替換、Emoji 替臉、即時手勢導播等專業版功能。
- 社群版使用獨立 bundle id `com.wondershow.community`，不會和主版權限身份混用。
- 打包產物包含簽名權限聲明，便於 macOS 正確請求攝影機、麥克風和螢幕錄製權限。
- 錄製主鏈路繼承 v1.0.0 受保護基線：活動視窗直接採集、切換錄製源、完整視窗顯示、預覽/匯出一致和高清匯出。

## 開源專案適合誰

- 想了解 WonderShow 專案格式、錄製 manifest、時間軸和匯出配置的開發者。
- 想寫 `.wondershow` 專案檢查器、轉換器、批次處理工具或素材管理工具的自動化開發者。
- 想接入本地 MediaPipe sidecar、實驗人像識別或手勢識別協議的研究者。
- 想開發第三方效果、輸入源、匯出整合或工作流插件的插件作者。
- 想學習 macOS 錄製產品如何拆分商業 App 與開放協議邊界的獨立開發者。

開源包不包含完整商業 App 源碼、ScreenCaptureKit 採集實現、即時監視器、影片合成器、商業 UI、授權系統、更新系統和專業版功能。它的目標是讓社群能基於公開協議做工具和整合，同時保留專業版繼續商業化的空間。
