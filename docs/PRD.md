# 产品需求文档

## 产品名称

- 中文：`灵演`
- 英文：`WonderShow`

## 产品定位

灵演是一款面向演讲、课程录制、直播和排练场景的 macOS 演示导演台。它负责把摄像头预览、演示控制、录制流程和手势输入整合为一套低学习成本的工作流。

## 核心用户

- 需要边讲边翻页的讲师
- 需要录制课程或演示视频的内容创作者
- 需要连接外接相机和大屏的直播/活动演讲者

## 当前问题

- 主界面更像研发调试台，缺少任务导向
- 手势误触较多，用户难以建立稳定肌肉记忆
- 诊断信息和主操作混在一起，普通用户不易上手

## 本阶段目标

- 把首页重构为“导演台”
- 把手势控制改成更稳的两阶段触发流程
- 保留后续 MediaPipe/sidecar 的扩展空间

## 非目标

- 本阶段不实现完整视频导出
- 本阶段不直接引入云端服务
- 本阶段不把 Figma 设计稿生成为仓库内静态资产

## 成功标准

- 用户进入首页 10 秒内能找到预览、彩排、录制和手势校准入口
- 手势翻页误触显著下降
- 界面能区分主任务区、状态区、诊断区
- 新架构支持后续替换底层手势引擎
