# WonderShow 灵演社区版

灵演社区版是面向创作者、讲师和开发者的免费可用版本。它保留了核心录制链路，让你可以在 macOS 上录制演示窗口、摄像头讲者画面和麦克风声音，并生成可预览、可导出的视频项目。

专业版仍在开发测试中。希望在不久的未来，专业版以及其他配套工具可以和大家见面。

## 功能介绍

- 摄像头接入：支持 Mac 内置摄像头、常见 UVC 摄像头和外接采集设备。
- 屏幕和窗口录制：可选择演示窗口、整个屏幕或可录制窗口源。
- 讲者画中画：支持 PPT/屏幕主画面加讲者画中画，也支持讲者主画面、左右分屏、只录屏、只录讲者等布局。
- 音频录制：跟随 macOS 当前麦克风输入，生成独立原始音频轨。
- 项目化保存：生成 `.wondershow` 项目结构，保留原始轨、合成轨、时间轴和导出信息。
- 合成预览与导出：录制后可以预览合成视频，并导出最终 MP4。
- 基础讲者画面调节：社区版保留镜像、亮度、对比、柔化等基础效果。
- 多语言界面：内置简体中文、繁体中文和英文界面。

## 应用场景

- 录制课程、微课、知识分享、产品演示和线上讲座。
- 给 Keynote、PowerPoint、WPS、PDF 或网页演示录制讲解视频。
- 为视频号、B 站、YouTube、内部培训平台准备带讲者画中画的素材。
- 在正式直播或录课前，验证摄像头、麦克风、屏幕录制权限和画面布局。
- 开发者验证 `.wondershow` 项目格式，编写项目读取、校验、转换或插件原型。

## 使用说明

1. 打开 `灵演社区版.app`。
2. 在 macOS 系统设置中允许摄像头、麦克风和屏幕录制权限。摄像头权限需要由 App 发起请求后才会出现在系统列表里。
3. 在右侧选择输入设备和音频输入。
4. 选择录制源：演示窗口、整个屏幕或手动选择窗口。
5. 选择录制模式和布局，例如“摄像头 + 屏幕”和“PPT主画面 + 讲者画中画”。
6. 点击“开始录制”，结束后等待合成输出。
7. 使用“预览合成”检查结果，再导出视频或保留项目文件继续处理。

## 特点说明

- 社区版强调稳定、可用和透明，不展示 VIP、SVIP 或手势控制入口。
- 社区版不包含正在实验中的高级美颜、背景替换、Emoji 替脸、实时手势导播等专业版功能。
- 社区版使用独立 bundle id `com.wondershow.community`，不会和主版权限身份混用。
- 打包产物包含签名权限声明，便于 macOS 正确请求摄像头、麦克风和屏幕录制权限。
- 录制主链路继承 v1.0.0 受保护基线：活动窗口直接采集、切换录制源、完整窗口显示、预览/导出一致和高清导出。

## 开源项目适合谁

- 想了解 WonderShow 项目格式、录制 manifest、时间轴和导出配置的开发者。
- 想写 `.wondershow` 项目检查器、转换器、批处理工具或素材管理工具的自动化开发者。
- 想接入本地 MediaPipe sidecar、实验人像识别或手势识别协议的研究者。
- 想开发第三方效果、输入源、导出集成或工作流插件的插件作者。
- 想学习 macOS 录制产品如何拆分商业 App 与开放协议边界的独立开发者。

开源包不包含完整商业 App 源码、ScreenCaptureKit 采集实现、实时监视器、视频合成器、商业 UI、授权系统、更新系统和专业版功能。它的目标是让社区能基于公开协议做工具和集成，同时保留专业版继续商业化的空间。
